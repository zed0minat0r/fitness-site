# Iron & Flow Fitness — Style Guide

## Colors
- Primary Dark: #1a1a1a
- Electric Orange: #FF6B35 (CTAs, accents)
- Clean White: #F5F5F5 (text on dark)
- Subtle Gray: #2a2a2a (cards)
- Text Secondary: #a0a0a0

## Typography
- Headings: Montserrat or Inter, 700-800 weight
- Body: Inter, 400 weight, 16px base
- Bold, energetic but readable

## Principles
- Dark theme default
- Mobile-first
- Energetic but not overwhelming
- No hover-only features
- Minimal tasteful animations
